Chatbot Docs

Steps to integrate chatbot in your website.

  • First create a account and login to conceptune console.

  • Create a project and select the project from the sidebar or dashboard which you have created.

  • In Api section of your project create and design emform only if you want to use emform in chatbot otherwise just create chatbot.

  • Now config emform only if you have created it. Also config chatbot with all the details given. Make sure if use emform you must check the checkbox that you are using emform in your chatbot while creating chatbot Api.

  • After creating Apis click on view option of chatbot api to get YOUR_PROJECT_ID, YOUR_CHATBOT_API_ID and YOUR_CHATBOT_API_KEY.

  • Add style sheet in your project HTML file.

  • Copy and paste this HTML code in your webpage and change the value of projectId, apiId and apiKey which are provided from conceptune console > project > api > chatbot > view

  • Reload your website and it's all Done. Your chatbot is integrated in few clicks.
